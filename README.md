# cheerstopaula.github.io
Template from [Theo Rashid](https://theorashid.github.io/site/home.html). Original README: Site generated using [Mizi](https://github.com/abstractxan/Mizi). Inspired by [hundredrabbits/100r.co](100r.co).

## run
- Update `src/config.conf` for webpages
- Update `src/website.md`
- Build using `./build.sh` or Run `./src/mizi` to create site
- Open `index.html` which opens `site/home.html`
